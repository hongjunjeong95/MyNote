

# Mongo Dump

```sh
mongodump --out <dumped_file_path>  --host <host_address> --port 27017 --db <db_name>
mongorestore --host <target host ip> --port 27017 <dumped_file_path>
```

## 아이디와 패스워드가 필요할 경우

```sh
mongodump
  --out mongo_local_dump  
	--host localhost
  --port 27017
  --username root
  --password 12345
  --authenticationDatabase admin
  --db <db_name>
  
mongorestore \
  --uri "mongodb+srv://{userName}:{password}@{host}/{dbName}?{queryParams}" \
  --drop \
  {dump_file_path}

mongorestore \
  --host {host} \
  --port 27017 \
  --username {userName} \
  --password {password} \
  --authenticationDatabase admin \
  --db <db_name> \
  /backup/mongoDump/myDB
```
